§ c4stream_install()

bool c4stream_install ( C4WriteStream *  ,
C4Error *   
)

Adds the data written to the stream as a finished blob to the store, and returns its key.

If you skip this call, the blob will not be added to the store. (You might do this if you were unable to receive all of the data from the network, or if you've called c4stream_computeBlobKey and found that the data does not match the expected digest/key.)
