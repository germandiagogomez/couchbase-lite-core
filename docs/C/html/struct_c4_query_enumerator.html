Detailed Description

A view query result enumerator.

Created by c4view_query, c4view_fullTextQuery, or c4view_geoQuery. Must be freed with c4queryenum_free. The fields of this struct represent the current matched index row, and are replaced by the next call to c4queryenum_next or c4queryenum_free. The memory pointed to by slice fields is valid until the enumerator is advanced or freed.
